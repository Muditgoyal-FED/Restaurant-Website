Definitions

  1. Website
    2.

    A place connected to the internet where a company, organization, etc. puts information that can be found on the World Wide Web.

  1. WWW
    2.  the full form of www is World wide web and also referred as w3 or web, the www is one of many applications
    built on top of the internet .it is an inter connected system of public webpages.
  1. HTTP/HTTPS
    2.

    HTTP stand for hypertext transfer protocol and HTTPs stands for hypertext transfer protocol secure.
    HTTP does not provide the security of the data but https is the secure version of HTTP protocol.

  1. Client server
    2.

    Client server is a relationship in which the program requests a service or resource from another program.

  1. Domain
    2.

    a domain is a address which allows a visitors to easily find the websites online.

  1. Host
    2.

    a host is a hardware that have a capability of permitting access to a network. A host work as a server offering information resources,
    services, applications to users or other hosts on the network.

  1. Hosting
    2.

    a hosting is a place where files are stored, its also known as web site hosting, web hosting and webhosting.

  1. FTP
    2.

    FTP is file transfer protocol and its basically allows the process of uploading and downloading files from a computer to the internet.

  1. Extension
    2.

    Extension tell the computer which application created or can open the file and which icon to use for the file.

  1. URL URI or URN
    2.

    URI stands for Uniform Resource Identifier. URL stands for Uniform Resource Location. URN stands for Uniform Resource Name. URI identifies the resource (text document, image file, etc).
    URL is the subset of URI that include a network location. URN is a subset of URI that include a name with in a given space, but no location.

  1. Address bar
    2.

    a address bar is also known as uniform resource locator an address bar is a component of an internet browser which is use to input and show the address of a website.
    The address bar helps the user in navigation by allowing entry of an internet protocol address or the url of a website.

  1. Title bar
    2.

    it is a horizontal bar at the top of the window, it displays the name an application defined icon and line of text. The text specifies the name of the application and indicates the purpose of the window.
    The title bar also makes it possible for the user to move the window using a mouse or other pointing device .
